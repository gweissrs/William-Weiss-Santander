# William Weiss — Agente Comercial Santander PJ

Landing page institucional para William Weiss, Agente Comercial Santander especializado em abertura de Contas PJ para MEIs e microempresas na Grande Florianópolis.

---

## Sobre o projeto

Site desenvolvido com foco em conversão, credibilidade e experiência do usuário. O objetivo é apresentar os serviços do William de forma clara, profissional e confiável — convertendo visitantes em clientes via WhatsApp.

---

## Stack

- **HTML5** semântico
- **CSS3** — variáveis, grid, flexbox, animações, responsividade
- **JavaScript** vanilla — sem frameworks ou dependências externas
- **Google Fonts** — Cormorant Garamond + Manrope

## Como usar

Não requer build ou instalação. Basta abrir o `index.html` em um navegador ou publicar os arquivos em qualquer hospedagem estática (GitHub Pages, Netlify, Vercel, etc.).

```bash
# Clonar o repositório
git clone https://github.com/seu-usuario/william-weiss.git

# Abrir no navegador
open index.html
```

---

## Decisões técnicas

- **Sem frameworks** — mantém performance máxima e zero dependências
- **CSS puro com variáveis** — sistema de design consistente e fácil de manter
- **IntersectionObserver** — animações de reveal sem impacto na performance
- **mix-blend-mode** — tratamento de logos com fundo preto sem edição de imagem
- **Cormorant Garamond + Manrope** — combinação editorial premium com legibilidade digital
